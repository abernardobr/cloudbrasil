     * All of the functions bellow will have callbacks for: validate, before, after

     *  createValidate: () => {},
     *  createBefore: () => {},
     *  createAfter: () => {}
     *
     * the paramentes are always:
     *      err: the first parameter is the error. If there is no error than it is null
     *      params: can be updated on each call to carry over data from one call to the other. At the end it will have retData, which is the data that is being returned

        Joi.assert(params, Joi.object().required());
